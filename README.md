## Системные требования:
- Windows 7 и новее
- Linux дистрибутивы 2013 года и новее
- OpenGL/ES 3.0

Поддержка с низким приоритетом:

- Adnroid 4.4+ (ARM-v7a/ARM-v8a/x86/x86_64)
